Transforming Mobile Money SMS into Meaningful Insights

About This Project

This platform processes and analyzes Mobile Money (MoMo) SMS data in XML format. The data is cleaned, categorized, and stored in a relational database before being presented here for visualization and analysis.

Our goal is to help users and organizations track financial activity, identify transaction trends, and make informed decisions from raw MoMo data.
